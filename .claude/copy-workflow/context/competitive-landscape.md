## Autodesk Forma — Detailed Positioning

**Their claim:** Cloud-based AI-powered early design — real-time environmental analysis (sun hours, daylight, wind, noise, embodied carbon) directly in the massing model.

**Their differentiators:**
- Seamless file-less sync with Revit
- Global scene coordinates via WGS84 for accurate placement
- Forma Site Design (pre-design feasibility) vs Forma Building Design (schematic)
- Global brand, trusted in large enterprises

**SAT vs Forma — how to position in copy:**
- Forma requires Autodesk enterprise subscription + existing Revit workflow → high barrier for Indian Tier-1 firms that are Revit-mixed but not Revit-only
- Forma has no IMD data integration → India climate data is generic interpolation, not meteorological station data
- Forma has no NBC 2016 / ECBC 2017 compliance layer
- Forma targets early schematic; SAT targets pre-design site selection + concept phase — can be used **before** Forma enters the workflow
- SAT is web-first, no install, no license prerequisite

**Copy positioning angle:**
> "Forma is the right tool once design starts. SAT is what you run before design starts — so every decision Forma helps you make is grounded in India-specific climate reality."

**Caution:** Never attack Autodesk directly by name in public copy. Frame as complementary workflow positioning.

## TestFit — Detailed Positioning

**Their claim:** Real estate feasibility platform — instantly generate building layouts that maximize pro forma yield.

**Their differentiators:**
- Fastest generative layout for parking + residential + mixed-use
- Financial modeling directly in layout tool
- Very strong in US market developer community

**SAT vs TestFit:**
- TestFit has zero environmental analysis — no solar, no wind, no flood risk, no climate data
- TestFit is US-centric; no Indian regulatory framework
- Different buyer: TestFit serves developers primarily; SAT serves architects
- Complementary, not competitive — SAT provides the environmental context that makes TestFit's layouts defensible to planners and end buyers

**Copy positioning angle:**
> "TestFit generates the layout. SAT tells you whether that layout will be liveable in Bengaluru in July — before your developer client asks."

## QGIS / ArcGIS — Detailed Positioning

**Their claim:** Professional-grade GIS desktop tools for geospatial analysis.

**SAT vs QGIS/ArcGIS:**
- Massive learning curve — designed for GIS professionals, not architects
- Requires data acquisition, layer management, plugin configuration — not plug-and-play
- Desktop software; no project collaboration, no Supabase-style persistence
- Zero AEC / design workflow integration

**Copy positioning angle:**
> "QGIS gives you the raw data if you know how to ask for it. SAT gives you the answer — in the language of design, not GIS."

## SAT's Differentiators (What No Competitor Has Together)

This is the set of claims that are defensible and unique to SAT. Use in positioning copy.

1. **India-native data**: IMD meteorological data, BBMP/BDA/NBC 2016/ECBC 2017 built-in. The only analysis platform with Indian regulatory vocabulary in its outputs.

2. **Architect-friendly, not GIS-expert-required**: Web-first, no install, results in 5 minutes. The UX is designed for architects, not data scientists.

3. **Full environmental stack in one tool**: Solar (pvlib + NREL SPA), flood risk (MERIT DEM + JRC), temperature (IMD + Open-Meteo), wind, rainfall, vegetation — no switching between 5 tools.

4. **Open formats**: GeoJSON + IFC exports, not proprietary. Revit-compatible handoff without translation.

5. **Pre-design, not schematic**: SAT is used before Forma, before Revit — at the site selection and concept phase. Fills a gap that competitors don't address.

6. **Cost**: Fraction of Autodesk enterprise licensing. Designed for 50–500 person firms, not global corporations.

## Competitor Copy Patterns to Avoid Mirroring

From analysis of Forma, TestFit, and AEC tech marketing:
- **Forma**: Heavy on "AI-powered" without specifics — SAT must be more precise about data sources
- **TestFit**: Pure financial framing — SAT's audience needs design + business combined
- **Generic AEC marketing mistake** (from Moncur research): "We poured the foundation" storytelling — project updates without meaning. SAT copy must communicate the architect's spatial philosophy, not just feature lists.
